# Advance Property Hub 🏠

A high-performance, full-stack Real Estate Management System designed to streamline property discovery and management. This project focuses on speed, security, and a premium user experience without the overhead of heavy frameworks.

## 🚀 Features
- **Advanced Search Engine:** Precision filtering by location, price, and property type.
- **Dynamic Listings:** Real-time property rendering directly from the database.
- **Powerful Admin Panel:** Comprehensive CRUD operations to add, edit, and delete listings.
- **Responsive UI:** A sleek, mobile-first design built with pure, custom CSS.
- **Secure Backend:** Robust data validation and optimized MySQL queries.

## 🛠️ Tech Stack
- **Frontend:** HTML5,JavaScript, Custom CSS (No Frameworks)
- **Backend:** PHP
- **Database:** MySQL
- **Environment:** XAMPP / WAMP
